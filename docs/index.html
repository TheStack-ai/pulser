Features

Diagnose. Prescribe. Fix.

📋

8 diagnostic rules

Frontmatter, description quality, file size, gotchas, allowed-tools, structure, conflicts, usage hooks — derived from Anthropic's published principles.

🔬

Auto-classifies skill type

Analysis, research, generation, execution, or reference — with confidence scoring. Prescriptions are tailored to each type.

💊

Prescriptions, not just warnings

Every issue comes with why it matters, what to add, and a ready-to-use template. Not "fix this" — "here's how and why."

🔧

--fix + pulser undo

Auto-apply safe structural fixes with full backup. Don't like the result? Roll back instantly.

🫀

Patient monitor TUI

EtCO2-style waveform animation in your terminal. Watch your skills get diagnosed in real time.

🤖

Claude Code skill built-in

Say "check my skills" or /pulser inside Claude Code. Claude runs the diagnosis and offers to fix issues conversationally.
